# 🚨 CrimeWatch AI — Real-Time Crime Detection System

> Government vehicle-mounted 360° camera system with instant AI-powered alerts

## Quick Start
```bash
pip install -r requirements.txt
python app.py
```

## Project Structure
```
crime-detection-system/
├── backend/
│   ├── ai/              # YOLOv8 detection engine
│   ├── api/             # FastAPI routes
│   ├── alerts/          # Telegram/Email alert system
│   └── database/        # SQLite models
├── frontend/
│   └── dashboard/       # React dashboard
├── simulation/          # Demo video simulation
├── app.py               # Main entry point
└── requirements.txt
```
